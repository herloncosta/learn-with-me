MDX é Markdown com a superpotência de usar **componentes** dentro do conteúdo.

## Por que MDX?

Com MDX você pode:

- Importar e usar componentes Astro/React/Vue/etc dentro do seu artigo
- Criar layouts customizados
- Reutilizar elementos complexos sem repetir HTML

### Exemplo

Você pode ter um código inline como `const x = 42` ou blocos destacados:

```ts
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

> MDX transforma seu conteúdo em componentes vivos, não apenas em texto estático.
